#### Git 기본 사용법

1. **clone** 로컬 저장소를 복제(clone)

   ```bash
   # 로컬 저장소를 복제(clone)
   $ git clone {/local/저장소/path}
   
   # 원격 서버의 저장소를 복제
   $ git clone {username@host:/원격/저장소/path}
   ```

2. **init** 프로젝트 폴더 초기화

   ```bash
   $ git init
   ```

3. commit

   ```bash
   $ git commit -m "{commit message}"
   ```

4. push

   ```bash
   $git push -u origin {branch name}
   ```
